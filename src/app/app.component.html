CSV Diff

This tool helps identify differences between two CSV files or spreadsheets.

Note: Security and confidentiality is important in this world, and this data is YOUR data. To that effect, No Data leaves your computer with this tool. All the work is done in your browser, and nothing is stored aside from you saving the final spreadsheet locally.

Status Column:
missing in X X had no matching row for the ID column, and therefore no value to compare.
empty in X X has a matching row for the ID column, but no value (or "NULL") for the column.
different Both data sources have a value (not missing or empty) and are different.
same Values for the columns in both data sources are equal. Note "NULL" is considered to be equal to having no value.
